# STM32 Motion Detection System

Motion detection using STM32L476RG and MPU6050 accelerometer

## Hardware
- STM32L476RG Nucleo board
- MPU6050 6-axis accelerometer/gyroscope
- I2C communication

## Features
- Real-time motion threshold detection
- Configurable sensitivity
- LED feedback for motion events
- UART output for sensor data monitoring

## Technical Details
- Bare-metal C programming
- I2C protocol implementation
- Sensor data processing and filtering
